# B站下载器 Skill

使用此skill从B站播放列表下载指定索引的MP3音频并生成SRT字幕文件。

## 功能说明

当用户提供B站视频URL和索引号时，执行以下步骤：

1. **下载MP3音频** - 使用yt-dlp从B站下载指定索引的音频
2. **生成SRT字幕** - 创建与音频文件同名的SRT字幕文件
3. **支持多种索引格式** - 单个索引、范围（如50-60）、列表（如1,3,5）

## 使用方式

当用户说类似以下内容时触发此skill：
- "下载B站第93首"
- "从B站下载https://... 索引94"
- "帮我下载B站播放列表第50-60首"
- "/bili-downloader <URL> <索引>"

## 执行步骤

### 1. 验证输入
- 检查是否提供了有效的B站链接
- 确认要下载的索引号

### 2. 下载音频
```bash
yt-dlp -x --audio-format mp3 -I <index> <bilibili_url>
```

### 3. 生成SRT字幕
- 为每个下载的MP3文件创建同名的SRT文件
- 如果用户提供了歌词，使用lyrics_to_srt.py转换
- 否则创建空的SRT占位文件

### 4. 完成交付
列出所有生成的文件并确认完成。

## 索引格式示例
- 单个: `93`
- 范围: `50-60`
- 列表: `1,3,5`
